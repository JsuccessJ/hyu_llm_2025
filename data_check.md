#  Sample data 데이터 확인

## 존재하는 key
- successful_count: 크롤링에 성공한 수
- failed_count: 크롤링에 실패한 수
- total_count: 총 크롤링 시도 수
- **perfumes_data: 향수 정보**
- failed_urls: 크롤링 실패한 url
